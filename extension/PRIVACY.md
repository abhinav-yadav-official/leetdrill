# LeetDrill Companion Privacy

LeetDrill Companion sends data only to https://abhiyadav.in/leetdrill.

The extension collects:

- LeetCode submission results from LeetCode problem pages.
- LeetCode cookies needed for authenticated LeetCode history import.
- The LeetDrill login cookie, used only to create an extension token for the same
  LeetDrill account.

The extension does not collect browsing history outside LeetCode problem pages,
does not inject advertising, and does not sell or share data with third parties.

Users can disconnect by removing the extension or clearing the extension token
from browser extension storage.
